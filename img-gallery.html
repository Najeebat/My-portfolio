Image Gallery

Project information

  • Category: Personal Projects
  • Project name: Image Gallery
  • Project date: Jan-2024
  • Project URL: https://najeebat.github.io/kodecamp-challenge-04-image-gallery/

The Image Gallery project is a visually captivating web application designed to showcase and organize a collection of images. Focusing on aesthetics and user experience, the gallery provides an engaging platform for users to explore and appreciate visual content. Whether used for personal portfolios, art displays, or corporate websites, the image gallery enhances the presentation of images in a cohesive and user-friendly manner.
Users can click on individual images to view them in a larger format without leaving the main gallery page.
